
### Sensory / Perceptual Modules
1. **Early Visual Processing** (V1–V4 like)  
   - **Major discovery**: Hubel & Wiesel's characterization of orientation-selective cells in primary visual cortex (V1), establishing feature detection hierarchy (1959–1962, Nobel 1981).

2. **Object/Face/Scene Recognition** (ventral stream / inferotemporal / fusiform / parahippocampal)  
   - **Major discovery**: Inferotemporal cortex lesions in monkeys impair object recognition (Klüver-Bucy syndrome extensions, 1930s–1950s); face-selective cells in IT cortex (Perrett et al., 1982); fusiform face area via fMRI (Kanwisher, 1997).

3. **Early Auditory Processing** (A1/core auditory cortex)  
   - **Major discovery**: Tonotopic organization in auditory cortex (Woolsey & Fairman, 1943–1950s); Ferrier's localization of audition in superior temporal gyrus (1881).

4. **Speech Sound / Phoneme Recognition** (superior temporal / Wernicke's-like)  
   - **Major discovery**: Carl Wernicke's description of sensory aphasia from posterior superior temporal damage (1874), linking region to language comprehension/phoneme processing.

### Motor / Action Modules
5. **Motor Planning & Execution** (premotor / primary motor / M1)  
   - **Major discovery**: Fritsch & Hitzig's electrical stimulation evoking movements from motor cortex (1870); Penfield's motor homunculus mapping in humans (1930s–1950s).

6. **Saccade / Gaze Control** (superior colliculus + frontal eye fields)  
   - **Major discovery**: Superior colliculus role in orienting/saccades (early 20th century); Ferrier's frontal eye field stimulation (1870s–1880s).

7. **Action Value / Policy Selection** (basal ganglia + prefrontal loops)  
   - **Major discovery**: Basal ganglia loops for action selection (Alexander et al., parallel circuits model, 1986); dopamine reward prediction error (Schultz, 1990s).

### Cognitive / Associative Modules
8. **Working Memory / Maintenance** (lateral PFC + parietal)  
   - **Major discovery**: Jacobsen's prefrontal lesions cause delayed-response deficits in monkeys (1936); Goldman-Rakic's PFC working memory model (1980s–1990s).

9. **Episodic Memory Encoding & Retrieval** (hippocampus + medial temporal)  
   - **Major discovery**: Scoville & Milner's case H.M. shows hippocampal/medial temporal lobe critical for new declarative/episodic memory (1957); place cells in hippocampus (O'Keefe, 1971, Nobel 2014).

10. **Semantic Memory / Conceptual Knowledge** (anterior temporal + ATL-like)  
    - **Major discovery**: Anterior temporal lobe hub for semantic processing (e.g., semantic dementia studies, Snowden et al., 1989–1990s; Lambon Ralph models, 2000s).

11. **Language Comprehension** (posterior temporal + Wernicke's-like)  
    - **Major discovery**: Carl Wernicke's area and sensory aphasia (1874).

12. **Language Production / Speech Planning** (Broca's + premotor)  
    - **Major discovery**: Paul Broca's frontal area linked to speech production (1861, from patient "Tan").

### Executive / Control / Integration Modules
13. **Salience / Attention Detection** (salience network: anterior insula + dorsal anterior cingulate)  
    - **Major discovery**: Insula/anterior cingulate activation in error/salience detection (early 1990s fMRI); Seeley et al. salience network definition (2007).

14. **Central Executive / Goal-Directed Control** (frontoparietal network / dorsolateral PFC)  
    - **Major discovery**: Duncan’s multiple-demand network (frontoparietal) for cognitive control (2010s); earlier PFC executive function from Phineas Gage (1848) and Luria's work (1960s–1970s).

15. **Default Mode / Internally Directed Thought** (DMN: posterior cingulate + medial PFC + angular gyrus)  
    - **Major discovery**: Raichle et al. identify default mode network during rest (2001).

### Affective / Motivational Modules
16. **Reward / Value Computation** (orbitofrontal + ventral striatum)  
    - **Major discovery**: Olds & Milner discover pleasure/reward centers (intracranial self-stimulation, 1954); Schultz dopamine neurons signal reward prediction error (1990s).

17. **Emotional Appraisal** (amygdala + insula)  
    - **Major discovery**: Kluver-Bucy syndrome shows amygdala role in emotion/fear (1930s); LeDoux's amygdala fear conditioning pathway (1980s–1990s).

### Multi-modal / Integrative Modules
18. **Multisensory Integration** (superior temporal sulcus + parietal)  
    - **Major discovery**: Superior temporal sulcus multisensory neurons (e.g., Bruce et al., 1981); parietal lobe integration (e.g., neglect syndromes, 1940s).

19. **Spatial Navigation / Egocentric → Allocentric Transformation** (hippocampus + retrosplenial + parietal)  
    - **Major discovery**: Hippocampal place cells (O'Keefe & Dostrovsky, 1971); grid cells in entorhinal cortex (Hafting et al., 2005, Moser Nobel 2014); head-direction cells (Taube, 1990).
